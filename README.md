## Install
```
$ git clone https://r7kamura@github.com/r7kamura/dotfiles.git
$ cd dotfiles

$ rake install
Create symlink: .gitconfig -> ~/.gitconfig
Create symlink: .gitexcludes -> ~/.gitexcludes
Create symlink: .gvimrc -> ~/.gvimrc
Create symlink: .inputrc -> ~/.inputrc
Create symlink: .pryrc -> ~/.pryrc
Create symlink: .tigrc -> ~/.tigrc
Create symlink: .tmux.conf -> ~/.tmux.conf
Create symlink: .vim -> ~/.vim
Create symlink: .vimrc -> ~/.vimrc
Create symlink: .zsh -> ~/.zsh
Create symlink: .zshrc -> ~/.zshrc
```


## Uninstall
```
$ rake uninstall
Remove symlink: .gitconfig -> ~/.gitconfig
Remove symlink: .gitexcludes -> ~/.gitexcludes
Remove symlink: .gvimrc -> ~/.gvimrc
Remove symlink: .inputrc -> ~/.inputrc
Remove symlink: .pryrc -> ~/.pryrc
Remove symlink: .tigrc -> ~/.tigrc
Remove symlink: .tmux.conf -> ~/.tmux.conf
Remove symlink: .vim -> ~/.vim
Remove symlink: .vimrc -> ~/.vimrc
Remove symlink: .zsh -> ~/.zsh
Remove symlink: .zshrc -> ~/.zshrc
```
